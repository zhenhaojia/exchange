### 方案优势
- ✅ **零配置** - 无需安装MySQL
- ✅ **自动初始化** - 自动创建数据库和表
- ✅ **包含测试数据** - 5个用户，5本图书
- ✅ **完整功能** - 支持所有核心功能

### 快速启动
```bash
# 进入项目目录
cd d:/桌面/exchange

# 启动SQLite版本（自动创建数据库）
npm run dev:sqlite
```

服务器将在以下地址运行：
- **前端**: http://localhost:5175
- **后端API**: http://localhost:3001

## 🛠 技术栈
- **前端**: React 18 + TypeScript + Vite + Material-UI
- **后端**: Node.js + Express
- **数据库**: SQLite（可切换MySQL）
- **认证**: JWT + bcrypt

## 🎉 开始使用

现在就运行命令启动系统：
```bash
npm run dev:sqlite
```

在浏览器中打开 http://localhost:5175 开始体验！
